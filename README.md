# Advanced Moderation Bot

An advanced Discord moderation bot designed for servers that need robust protection. Features include:

- **Automated moderation** (detects spam, raid attempts, and malicious links)  
- **Logging system** (tracks deleted messages, joins/leaves, and more)  
- **Customizable commands** for moderators  
- **Raid detection & prevention**  
- **User-friendly setup** and configuration  

This bot is ideal for Discord communities that want **smarter, safer moderation** without constant manual oversight.

## Features
- Anti-spam & anti-raid protection  
- Role-based permissions  
- Event logging (joins, leaves, message deletions)  
- Warning and kick/ban system  
- Easy to configure and extend  

## Installation
1. Clone the repository  
2. Install dependencies (`npm install` for Discord.js, etc.)  
3. Configure your `config.json` with bot token & server settings  
4. Run with `node index.js`
